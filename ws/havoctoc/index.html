Microsoft Edge is required. This tool only works as intended in Microsoft Edge.

  1. Open the page in Microsoft Edge.
  2. In Edge settings, click Read Aloud once then stop it, to initialize Edge's Text-to-Speech engine. If successful, Microsoft Steffan should appear as the selected default voice.
  3. Hold the Push-to-Talk radio button and transmit naturally, for example:
    "Havoc, we have eyes on the HVT. What's the intent on target, over?"
  4. HAVOC (the fictional TOC Commander) will respond with dynamic orders or information generated in real time using Gemini AI, providing immersive tactical radio roleplay.

Responses are AI-generated and may occasionally make mistakes, but they are designed to enhance mission immersion and communications roleplay.

Enjoy, operators.
